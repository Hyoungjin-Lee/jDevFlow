## Sec. 2. F-D3 19명 표시 layout (Q1 흡수, R-1·R-8 정정)

### 2.1 19명 표시 산식 (R-1·R-8 본문 정정)

운영자 결정(Q1 = F-D3 final)과 4팀 인원 정합을 위한 본문 산식:

```
19명 표시 = 박스 영역 (4팀 15명) + PM 상단 status bar (1명)
         + 미표시 (CTO 백현진 + CEO 이형진 = 2명, 자기 자신 비표시)
         + placeholder (HR팀 미배정 1명, 향후 추가 슬롯)
         = 조직 분량 19명
```

| 구분 | 인원 | 표시 방식 |
|------|------|----------|
| 박스 영역 (4팀) | **15명** (기획4 + 디자인4 + 개발7) | 박스 3개 안 행 단위 표시 |
| PM 스티브 리 | **1명** | 상단 `Header` status bar 1행 (PMStatusBar 위젯) |
| CTO 백현진 | **1명** | 미표시 (회의창 = CTO 자기 자신 비표시) |
| CEO 이형진 | **1명** | 미표시 (운영자 = CEO 자기 자신 비표시) |
| HR팀 미배정 | **1명** | placeholder (향후 추가 슬롯, `personas_18.md` 영역) |
