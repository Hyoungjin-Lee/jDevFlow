작업 방식:
1. 위 파일들을 읽고 현재 프로젝트 목적과 작업 순서를 복원해라.
2. `2026-04-21-claude-execution-sequence.md` 기준으로 현재 수행해야 할 phase를 확인해라.
3. 기본 시작점은 Phase 1이다.
4. 따라서 특별한 지시가 없으면 먼저 Phase 1 handoff와 execution prompt를 읽고 workflow architecture 작업부터 시작해라.
5. 각 phase는 순차적으로 수행하고, 이전 phase 결과를 다음 phase의 기준으로 삼아라.

기본 시작 규칙:
- 특별한 추가 지시가 없다면 Phase 1부터 시작
- Phase 1 완료 후 결과를 정리
- 이어서 Phase 2
- Phase 2 후 실제 반영 범위를 기준으로 Phase 2B attribution 정리

완료 후 응답:
- 지금 어떤 phase를 시작했는지
- 어떤 파일을 읽었는지
- 어떤 문서를 기준으로 작업에 들어갔는지
- 실제 수정한 파일 목록
- 다음 phase로 어떻게 이어질지

작업을 멈추지 말고, 문서 기반으로 현재 상태를 복원한 뒤 실제 파일 수정 작업까지 진행해라.
