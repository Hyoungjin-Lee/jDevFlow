### M1-1. `scripts/monitor_bridge.sh` 신규 작성

design_final Sec.5.3 (F-62-7 a/b/c) + Sec.3.1 + Sec.6.1 (Q1) 따라:

```bash
#!/usr/bin/env bash
# 환경 변수 주입 (Q1 패턴)
target_session="${TMUX_SESSION_BRIDGE:-bridge}"

prev=""
while true; do
  # F-62-7c: -S -20 범위 한정
  cur=$(tmux capture-pane -t "$target_session" -p -S -20 2>/dev/null \
    | grep --line-buffered -E "^📡 status|^ERROR|^운영자 결정|중단 조건|FAIL|S[0-9]+ ✅" \
    | tail -5)
  if [ "$cur" != "$prev" ] && [ -n "$cur" ]; then
    # F-62-7a/b: timestamp + stage 명시
    timestamp=$(date +%Y-%m-%d\ %H:%M:%S)
    echo "📡 status $timestamp $cur"
    prev="$cur"
  fi
  sleep 3
done
```

요건: ≥ 30줄, `📡 status` / `-S -20` / `date +%Y-%m-%d` 패턴 필수.
