# 대시보드 토큰 누적값 표시 — Stop hook 연결

## 배경

v0.6.4 대시보드가 토큰 값을 읽지 못하는 원인:
- `.claude/dashboard_state/` 디렉토리 없음
- Claude Code `Stop` hook이 settings.json에 없음 → 응답 종료 시 토큰 데이터를 파일로 안 씀
- `TokenHook`의 1순위 경로(JSON 파일)가 동작 불가 → 항상 0.0k 표시

기존 코드 (`scripts/dashboard/token_hook.py`) 는 이미 `dashboard_state/` 폴더 내 JSON 파일을 읽도록 구현돼 있음. 파일만 생기면 바로 작동함.

## 작업 내용

### S1 — dashboard_state 폴더 생성

```
.claude/dashboard_state/
.claude/dashboard_state/.gitkeep
```

`.gitignore`에 `*.json` 패턴 추가 (토큰 데이터 파일은 커밋 제외, `.gitkeep`만 추적).

### S2 — Stop hook 스크립트 작성

`scripts/hook_stop.sh` 신규 작성.

Claude Code `Stop` hook이 stdin으로 넘겨주는 JSON 구조:
```json
{
  "session_id": "...",
  "transcript_path": "...",
  "usage": {
    "input_tokens": 1234,
    "output_tokens": 567,
    "cache_creation_input_tokens": 0,
    "cache_read_input_tokens": 0
  }
}
```

스크립트가 할 일:
1. stdin에서 JSON 읽기
2. `session_id` + `usage` 추출
3. PROJECT_ROOT 기준 `.claude/dashboard_state/` 에 `joneflow_<project_hash>_<session_id>.json` 파일 저장 (token_hook.py의 네이밍 규칙과 정합)
4. 누적 방식: 파일이 이미 있으면 `input_tokens` + `output_tokens` 합산해서 덮어쓰기
5. 실패 시 `exit 0` 강제 (hook은 차단하면 안 됨)

파일 형식 (token_hook.py REGEX 정합):
```json
{"usage": {"input_tokens": 1234, "output_tokens": 567}}
```

### S3 — settings.json Stop hook 등록

`hooks` 객체 안에 `Stop` 이벤트 추가:
```json
"Stop": [
  {
    "hooks": [
      {
        "type": "command",
        "command": "scripts/hook_stop.sh"
      }
    ]
  }
]
```

### S4 — 검증

1. `python3 -m py_compile scripts/hook_stop.sh` — 해당 없음 (bash)
2. `bash -n scripts/hook_stop.sh` — 문법 검사
3. echo 시뮬레이션:
   ```bash
   echo '{"session_id":"test-001","transcript_path":"","usage":{"input_tokens":1000,"output_tokens":200,"cache_creation_input_tokens":0,"cache_read_input_tokens":0}}' | bash scripts/hook_stop.sh
   cat .claude/dashboard_state/joneflow_*test-001*.json
   ```
4. token_hook.py import 테스트:
   ```bash
   venv/bin/python3 -c "
   from scripts.dashboard.token_hook import TokenHook
   from scripts.dashboard.tmux_adapter import TmuxAdapter
   th = TokenHook(project_root=__import__('pathlib').Path('.'))
   print('import OK')
   "
   ```

## 완료 조건

- `git commit` (scripts/hook_stop.sh + .claude/settings.json + .claude/dashboard_state/.gitkeep + .gitignore 수정)
