# 테스트 작업 — 대시보드 실행 검증

## 목적

회의창 → bridge → Orc → 팀원 파이프라인 동작 확인용 간단 작업.

## 작업 내용

1. `venv/bin/python3 -c "from scripts.dashboard import dashboard; print('import OK')"` 실행해서 대시보드 모듈 import 정상 여부 확인
2. `venv/bin/python3 -m py_compile scripts/run_dashboard.py scripts/dashboard/dashboard.py scripts/dashboard/render.py scripts/dashboard/data.py scripts/dashboard/notifier.py` 실행해서 문법 오류 없는지 확인
3. 결과를 📡 status: PASS 또는 FAIL로 보고

## 완료 조건

- import OK + py_compile 오류 0건 → PASS
- 오류 발견 시 오류 메시지 포함해서 FAIL 보고
